# Android Applications

A collection of Android apps built with Kotlin, Java, and XML using OOP principles.

## Apps

### 🎨 Kids Drawing App
- Interactive canvas using Android Canvas API and Touch Events
- Supports freehand drawing and custom brush tools
- **Tech:** Kotlin, XML, Android Canvas API

### ❓ Quiz App
- Modular data classes for question management using OOP
- Efficient state handling throughout the app
- **Tech:** Kotlin, Java, XML

### 🔢 Calculator App
- Basic arithmetic operations with an intuitive UI
- **Tech:** Kotlin, XML

## Tech Stack
Kotlin · Java · XML · Android Studio · OOP
